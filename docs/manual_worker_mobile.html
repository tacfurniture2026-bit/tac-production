📱 アプリとしてホーム画面に追加する方法

QRコードをスキャンしてサイトを開いたら、以下の手順でアプリとして追加すると便利です。

🍎 iPhone (iOS) の場合

1Safari でサイトを開く（上のQRコードをスキャン）

2画面下部の 「共有」ボタン (□から↑が出ているアイコン) をタップ

3メニューを少し下にスクロールして 「ホーム画面に追加」 をタップ

4右上の 「追加」 をタップ

✅ ホーム画面にアイコンが追加されます！

🤖 Android の場合

1Chrome でサイトを開く（上のQRコードをスキャン）

2右上の 「︙」メニュー をタップ

3「ホーム画面に追加」 または 「アプリをインストール」 をタップ

4確認画面で 「追加」 をタップ

✅ ホーム画面にアイコンが追加されます！

🔐 ログイン方法

  1. 追加したアプリアイコンをタップして開く
  2. ユーザー名・パスワードを入力
    • 管理者: admin / admin123
    • 作業者: worker / worker123
  3. 「ログイン」 をタップ

📦 棚卸スキャンの使い方

基本の流れ

1画面下部の 「棚卸」 アイコンをタップ

2「📷 カメラ起動」 をタップ（初回は「許可」を選択）

3商品の QRコード をカメラで映す

4商品情報が表示されたら、実棚数量 を入力

5「登録」 をタップ

✅ 登録完了後、自動でカメラが再起動します（連続スキャン可能）

手動入力（QRがない場合）

1「手動入力」ボタンをタップ

2資材IDを直接入力

3数量を入力して「登録」

🔍 在庫検索の使い方

  1. 画面下部の 「検索」 アイコンをタップ
  2. 品名やIDの一部を入力して検索
  3. 現在の在庫数・金額リストが表示されます

↕️ 在庫増減（入庫・出庫）の使い方

  1. 画面下部の 「在庫増減」 アイコンをタップ
  2. 資材IDを入力
  3. 「入庫」 または 「出庫」 を選択
  4. 数量を入力（メモがあれば記入）
  5. 「登録」 をタップ

❓ よくある質問 (FAQ)

Q. カメラが起動しません
A. ブラウザのカメラ許可設定を確認してください。
iPhone: 設定 → Safari → カメラ → 「許可」
Android: 設定 → アプリ → Chrome → 権限 → カメラ → 「許可」
Q. QRコードが読み取れません
A. 以下を試してください：
  • 明るい場所で撮影する
  • QRコードを画面中央に合わせる
  • カメラを近づけすぎず、少し離してみる
Q. データが保存されません
A. インターネット接続を確認してください（Wi-Fiまたはモバイルデータ通信）。
