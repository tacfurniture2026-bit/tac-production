**https://tacfurniture2026-bit.github.io/tac-production/**

3. 「ログイン」をクリック

---

## 📦 在庫管理メニュー

サイドバーの「在庫管理」セクションから各機能にアクセスできます。

| メニュー | 機能 |
|----------|------|
| 棚卸スキャン | QRコードで棚卸登録 |
| 在庫検索 | 在庫状況の確認 |
| 商品マスタ | 商品の登録・編集・削除 |
| 在庫増減 | 入庫・出庫の手動登録 |
| 月次締め | 月次在庫金額の確定 |

---

## 1. 商品マスタ

### 商品を登録する
1. 「商品マスタ」を開く
2. 「+ 新規登録」ボタンをクリック
3. 以下を入力：
   - 資材ID（自動採番可）
   - 品名
   - 分類
   - 単価
   - 不動品の場合はチェック
4. 「保存」

### CSV一括取り込み
1. スプレッドシートから「ファイル > ダウンロード > CSV」
2. 「📥 CSV取込」をクリック
3. ダウンロードしたCSVファイルを選択
4. プレビューを確認して「取込実行」

> 既存IDは更新、新規IDは追加されます

### CSV出力
1. 「📤 CSV出力」をクリック
2. CSVファイルがダウンロードされる

---

## 2. 在庫増減

入庫・出庫を手動登録します。

1. 「在庫増減」を開く
2. 資材IDを入力
3. 「入庫」または「出庫」を選択
4. 数量とメモを入力
5. 「登録」

---

## 3. 棚卸スキャン

QRコードで棚卸を行います。

1. 「棚卸スキャン」を開く
2. 「📷 カメラ起動」をクリック
3. 商品のQRコードをスキャン
4. 実棚数量を入力
5. 「登録」

---

## 4. 在庫検索

現在の在庫状況を確認します。

1. 「在庫検索」を開く
2. キーワードを入力して検索
3. 在庫数・単価・金額が一覧表示

---

## 5. 月次締め

月次の在庫金額を確定します。

### 集計確認
1. 「月次締め」を開く
2. 対象年月を選択
3. 「📊 集計確認」をクリック
4. サマリーカードで確認：
   - 📦 在庫金額合計
   - ✓ 通常在庫
   - ⚠ 不動品在庫
   - 💰 TAC口銭込（×1.01）

### 月次締め実行
1. 集計内容を確認
2. 「🚀 月次締め実行」をクリック
3. 確認で「OK」→ データ確定

---

## 6. 月次報告

生産データと在庫データを統合した報告書です。

1. 「月次報告」を開く
2. 期間を設定して「集計」
3. 表示内容：
   - 月次総括サマリ
   - **在庫金額サマリ（通常/不動品/グラフ）**
   - 詳細データ一覧
   - 部門別集計

---

## 💡 用語説明

| 用語 | 説明 |
|------|------|
| 不動品 | 長期間動かない在庫。月次集計で別集計 |
| TAC口銭 | 在庫金額の1%を加算した金額 |
| リアルタイム同期 | 複数端末でデータが自動同期 |
